# Padaria Braga — Landing Page Premium

Layout **editorial premium** para a **Padaria Braga** (bairro Industrial, Contagem-MG). Design distinto, pensado para portfólio.

**Site no ar:** https://eukevytosdev.github.io/padaria-braga-landing/

## Stack

HTML · CSS · JavaScript puro (sem framework)

## Estrutura

```
├── index.html
├── css/
│   ├── reset.css
│   ├── variables.css   # tokens de design
│   └── style.css       # layout editorial premium
├── js/
│   ├── config.js       # dados da padaria (edite aqui)
│   └── main.js
└── assets/images/
    └── logo-placa.png
```

## Destaques do layout

- Hero assimétrico com tipografia grande (Cormorant Garamond + Outfit)
- Marquee animado com categorias
- Grid bento de produtos com hover
- Banner full-bleed + lista numerada de experiências
- Avaliações estilo Google com score editorial
- Card flutuante sobre mapa no contato

## Editar conteúdo

| Arquivo | O que mudar |
|---------|-------------|
| `js/config.js` | Textos, WhatsApp, imagens, produtos, avaliações |
| `css/variables.css` | Cores e tokens de design |

## Abrir localmente

Abra `index.html` no navegador ou use **Live Server**.
